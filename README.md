# sergeroberge.com

Source for the personal site of Serge Roberge.

This site presents operating perspectives on engineering leadership,
platform modernization, and technology execution in environments where
delivery expectations exceed existing systems or organizational capacity.

The site is intentionally static and minimal. Content is maintained as
long-form written material rather than blog or marketing content.

## Structure

- Astro static site
- Markdown-authored content
- Deployed via GitHub Pages
- Fronted by Cloudflare

## Purpose

The repository exists as a durable, versioned record of professional
operating philosophy and experience.

Most professional work referenced on the site is conducted within
private enterprise environments and is therefore not publicly visible.

## Status

Actively maintained.
